<summary><strong>Unit 2: Algorithms</strong></summary>

### Computational Thinking
- Breaking large problems into smaller, manageable pieces
- Thinking step-by-step like a computer

### Algorithms
- A clear set of instructions to solve a problem
- Must be ordered, specific, and finite

### Pseudocode
- Writing code logic using plain English
- Helps plan programs without worrying about syntax

### Flowcharting
- Visual diagrams that show the steps of an algorithm
- Use shapes to represent actions and decisions

### What is Abstraction?
- Removing unnecessary details to focus on the main idea
- Makes complex systems easier to understand

</details>

<details>
<summary><strong>Unit 3: Data</strong></summary>

### Data Overview
- Data is information collected for analysis or decision-making
- Can be numbers, text, images, or sound

### Data Systems
- Tools used to store and manage data
- Examples include databases and spreadsheets

### Data Abstraction
- Simplifying data to highlight important patterns
- Helps manage large data sets

### Digital Data Representation
- All data is stored using binary (0s and 1s)
- Different formats represent different types of data

### Binary Number System
- Base-2 number system using only 0 and 1
- Each digit represents a power of 2

### Binary Search vs Linear Search
- Linear search checks each item one by one
- Binary search repeatedly splits a sorted list (faster)

### Searching & Sorting
- Searching finds data
- Sorting organizes data to make searching faster

### ASCII
- A system that assigns numbers to letters and symbols
- Allows text to be stored digitally

### Data Compression
- Reduces file size
- Can be lossless or lossy

### Data Compression and Storage
- Smaller files save space and transmit faster
- Tradeoffs exist between quality and size

</details>

<details>
<summary><strong>Unit 10: The Internet</strong></summary>

### The Internet Overview
- A global network of connected computers
- Allows information to be shared worldwide

### History of the Internet
- Started as a research project
- Expanded into a global system

### Internet Systems
- Includes servers, clients, and routers
- Work together to transmit data

### Using HTML to Create a Webpage
- HTML structures content on the web
- Uses tags for text, images, and links

### What is the Internet?
- A network of networks
- Uses protocols to communicate

### Building a Network
- Devices connected to share data
- Can be wired or wireless

### The Need for Addressing
- Devices need unique addresses (IP)
- Ensures data reaches the right location

### Routing & Redundancy
- Routing determines data paths
- Redundancy prevents system failure

### Packets
- Data is split into small pieces
- Reassembled at the destination

### HTTPS & DNS
- HTTPS secures web communication
- DNS converts domain names to IPs

### Fault Tolerance
- Systems continue working despite failures
- Important for reliability

### Parallel and Distributed Computing
- Parallel: tasks run at the same time
- Distributed: tasks spread across computers

</details>

<details>
<summary><strong>Unit 11: Cybersecurity</strong></summary>

### Cybersecurity Overview
- Protecting systems and data
- Prevents attacks and data theft

### PIIs & Cookies
- PII: Personally Identifiable Information
- Cookies store small user data

### Internet Privacy
- Concerns about how data is collected and used
- Users should be aware of tracking

### Digital Footprint
- The data you leave online
- Can be permanent

### Passwords
- Protect accounts from unauthorized access
- Strong passwords are long and unique

### Cybersecurity & Cryptography
- Cryptography secures information
- Uses math to protect data

### Cryptography
- Encoding information to keep it secret
- Only authorized users can read it

### Encryption
- Converts readable data into unreadable form
- Requires a key to decrypt

</details>

<summary><strong>Unit 12: Impact of Computing</strong></summary>

### Impact of Computing Overview
- Computing affects society and culture
- Can be both positive and negative

### Machine Learning
- Computers learn patterns from data
- Used in recommendations and predictions

### Training Data & Bias
- Models learn from provided data
- Biased data creates biased outcomes

### Algorithmic Bias
- Algorithms may unfairly favor groups
- Important ethical concern

</details>
